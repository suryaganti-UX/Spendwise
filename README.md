# Spendwise

Spendwise is a privacy-first personal finance web app that helps users understand where their money went by parsing Indian bank statement PDFs directly in the browser.

## Features

- Upload bank statement PDFs
- Local/browser-only processing
- Category breakdowns
- Monthly trends
- Top merchants
- Recurring charges insights
- Demo/sample data flow

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript
